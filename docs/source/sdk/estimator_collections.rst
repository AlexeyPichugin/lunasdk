Estimator collections
=====================


99% real applications is needed one instance of faceengine and one instance of every estimators. Initialization of
estimators is to slow therefore application should initialize all needing estimators on start up and re use its.

A estimator collection are a logical groped estimators. Estimators from collection are initialized in first call.

Class *FaceEstimatorsCollection* contains all estimators for a face.
