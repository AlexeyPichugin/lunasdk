Configuration
=============


All default settings is stored in configuration file *"faceengine.conf"*. Folder *"$FSDK_ROOT/data" contains this file.
You can set up own *"faceengine.conf"* in a *VLFaceEngine* initialization. You can find a detailed information about
settings in *"ConfigurationGuide.pdf"*.
